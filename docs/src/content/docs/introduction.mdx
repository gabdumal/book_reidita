Este livro pretende ensinar a criar um jogo do gênero _Idle Clicker_ para a Web.
Utilizaremos as seguintes tecnologias:

- **HTML**: linguagem de marcação para a Web
- **CSS**: linguagem de estilização para a Web
- **Typescript**: _superscript_ da linguagem de programação Javascript
- **React**: biblioteca de desenvolvimento de interfaces para a Web

## Jogos Idle Clicker

O gênero _Idle Clicker_ é um tipo de jogo que se popularizou na Web.
Seus fundamentos são simples: o jogador clica em um botão para ganhar algum tipo de ponto.
Com esses pontos, o jogador pode comprar melhorias que aumentam a quantidade de pontos ganhos por clique ou por segundo.

O objetivo é avançar no jogo para descobrir novas mecânicas e melhorias, além de possivelmente eventos da história do jogo.

Um exemplo de jogo _Idle Clicker_ é o [Cookie Clicker](https://orteil.dashnet.org/cookieclicker/).
Seu tema é uma confeitaria de biscoitos, em que o jogador clica em um _cookie_ para ganhar mais _cookies_.
O desenvolvimento da história apresenta diversos minigames e mecânicas.

## HTML

HTML significa _HyperText Markup Language_, ou _Linguagem de Marcação de Hipertexto_.
Ela é responsável por estruturar o conteúdo de uma página, desde seus metadados até o texto e demais mídias que a compõem.

Este é um exemplo de um documento HTML:

```html
<!doctype html>
<html lang="pt-BR">
  <head>
    <meta charset="UTF-8" />
    <title>Meu Jogo</title>
  </head>
  <body>
    <h1>Meu Jogo</h1>
    <p>Olá, mundo!</p>
  </body>
</html>
```

O HTML é composto por _tags_ que indicam o início e o fim de elementos.
No exemplo acima, a tag `<p>` indica que seu conteúdo é um texto comum.

Podemos aninhar tags em outras, como a tag `<body>`, que contém as `<h1>` e `<p>`.
Essa é responsável por definir todo o conteúdo visível do site.

A tag `<head>`, por outro lado, contém informações sobre o documento, como o título da página e metadados.
Nesse exemplo, especificamos a codificação de caracteres do documento com a tag `<meta charset="UTF-8" />`.

Toda a página deve ficar dentro da tag `<html>`, que define o início e o fim do documento.
Para identificar o seu tipo, usamos a tag especial `<!doctype html>`.

## CSS

CSS significa _Cascading Style Sheets_, ou _Folhas de Estilo em Cascata_.
Essa linguagem é responsável por estilizar o conteúdo de uma página HTML.

Fazemos isso associando regras de estilo a elementos HTML.
Por exemplo, para mudar a cor do texto de um parágrafo, podemos usar a seguinte regra:

```css
p {
  color: blue;
}
```

Há diferentes formas de adicionar esse texto à página.
Vamos fazer isso ao utilizar uma tag chamada `<style>` dentro da tag `<head>` do documento HTML:

```html
<head>
  <!-- ... -->
  <style>
    p {
      color: blue;
    }
  </style>
</head>
```

## Javascript

Javascript é uma linguagem de programação que permite adicionar interatividade a páginas Web.

Podemos utilizá-la para adicionar comportamentos dinâmicos, como alterar o conteúdo de uma página ou responder a eventos do usuário.

O código abaixo adiciona um evento de clique a um parágrafo, exibindo um alerta ao usuário:

```javascript
const paragraph = document.querySelector("p");

paragraph.addEventListener("click", () => {
  window.alert("Você clicou no parágrafo!");
});
```

Para adicionar esse código a um documento HTML, podemos usar a tag `<script>` dentro da tag `<body>`.

```html
<body>
  <!-- ... -->
  <script>
    const paragraph = document.querySelector("p");

    paragraph.addEventListener("click", () => {
      window.alert("Você clicou no parágrafo!");
    });
  </script>
</body>
```
