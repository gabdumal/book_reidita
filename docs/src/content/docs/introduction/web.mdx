A Web é um ambiente vasto e diversificado.
Seu conteúdo é majoritariamente composto por páginas, que podem variar de simples textos a jogos complexos.
Para criar uma página Web, utilizamos diferentes tecnologias, como HTML, CSS e JavaScript.

## HTML

HTML significa _HyperText Markup Language_, ou _Linguagem de Marcação de Hipertexto_.
Ela é responsável por estruturar o conteúdo de uma página, desde seus metadados até o texto e demais mídias que a compõem.

Este é um exemplo de um documento HTML:

```html
<!doctype html>
<html lang="pt-BR">
  <head>
    <meta charset="UTF-8" />
    <title>Meu Jogo</title>
  </head>
  <body>
    <h1>Meu Jogo</h1>
    <p>Olá, mundo!</p>
  </body>
</html>
```

O HTML é composto por _tags_ que indicam o início e o fim de elementos.
No exemplo acima, a tag [`<p>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/p) indica que seu conteúdo é um texto comum.

Podemos aninhar tags em outras, como a tag [`<body>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/body), que contém as [`<h1>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/h1) e `<p>`.
Essa é responsável por definir todo o conteúdo visível do site.

A tag [`<head>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/head), por outro lado, contém informações sobre o documento, como o título da página e metadados.
Nesse exemplo, especificamos a codificação de caracteres do documento com a tag [`<meta charset="UTF-8" />`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/meta).

Toda a página deve ficar dentro da tag [`<html>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/html), que define o início e o fim do documento.
Para identificar o seu tipo, usamos a tag especial `<!doctype html>`.

## CSS

CSS significa _Cascading Style Sheets_, ou _Folhas de Estilo em Cascata_.
Essa linguagem é responsável por estilizar o conteúdo de uma página HTML.

Fazemos isso associando regras de estilo a elementos HTML.
Por exemplo, para mudar a cor do texto de um parágrafo, podemos usar a seguinte regra:

```css
p {
  color: blue;
}
```

Há diferentes formas de adicionar esse texto à página.
Vamos fazer isso ao utilizar uma tag chamada [`<style>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/style) dentro da tag `<head>` do documento HTML:

```html
<head>
  <!-- ... -->
  <style>
    p {
      color: blue;
    }
  </style>
</head>
```

## JavaScript

JavaScript é uma linguagem de programação que permite adicionar interatividade a páginas Web.

Podemos utilizá-la para adicionar comportamentos dinâmicos, como alterar o conteúdo de uma página ou responder a eventos do usuário.

O código abaixo adiciona um evento de clique a um parágrafo, exibindo um alerta ao usuário:

```javascript
const paragraph = document.querySelector("p");

paragraph.addEventListener("click", () => {
  window.alert("Você clicou no parágrafo!");
});
```

Para adicionar esse código a um documento HTML, podemos usar a tag [`<script>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/script) dentro da tag `<body>`.

```html
<body>
  <!-- ... -->
  <script>
    const paragraph = document.querySelector("p");

    paragraph.addEventListener("click", () => {
      window.alert("Você clicou no parágrafo!");
    });
  </script>
</body>
```
